# 一.赋值推断

赋值时推断，类型从右像左流动,会根据赋值推断出变量类型

let str = 'ailjc';
let age = 11;
let boolean = true;

# 二.返回值推断

自动推断函数返回值类型

  return a + b;
}

# 四.属性推断

可以通过属性值,推断出属性的类型

let person = {
  name: 'zf',
  age: 11,
};
let { name, age } = person;

  age: 11,
